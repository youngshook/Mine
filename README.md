Mine
=========

This is a personal project, an app with a Parse Backend.

***Features:***

* Multiple VC
* Parse Database
* User authentication
* Badge number

***Planned features:***

* Detailed Item visualization
* Edit Item from Detaled Item Visualization
* Other's items visualization ("Contacts")
* Accept/decline contacts
